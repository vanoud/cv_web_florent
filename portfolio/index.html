  • my_paint

    Ce projet consiste à réaliser un "paint-like" en JavaScript à l’aide du composant HTML5 "canvas". La taille de la zone de travail n’a pas d’importance. Vous pouvez choisir celle que vous voulez. Comme tous les "logiciels" de dessin, votre my_paint devra avoir, au minimum les outils indiqués ci-dessous : • Le "crayon" : ajoute un pixel quand on clique ou que l’on reste appuyé sur le bouton de la souris. • La "ligne" : premier click pour l’origine, deuxième pour la destination. • Le "rectangle" : premier click pour l’origine, deuxième pour la taille. • Le cercle : premier click pour le centre, deuxième pour le rayon. La gestion des couleurs doit également être eﬀective (à vous de trouver le moyen de les selectionner, les représenter, etc...). Bon projet !
