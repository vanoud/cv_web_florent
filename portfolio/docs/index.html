Introduction

Ergo is a template based on a modified version of Bootstrap framework from Twitter. This version has removed the border-radius property for all components and also the gradients background has been changed to solid colors, giving a flat style to all stantard components.

CSS

Files

The theme use the following cascade stylesheets

  • bootstrap.css: Bootstrap main styles unminified
  • bootstrap.min.css: Bootstrap main styles minified
  • bootstrap-responsive.css: Bootstrap responsive styles unminified
  • bootstrap-responsive.min.css: Bootstrap responsive styles minified
  • ergo.css: Ergo theme main styles unminified
  • ergo.min.css: Ergo theme main styles minified
  • font-awesome.css: Font Awesome main styles unminified
  • font-awesome.min.css: Font Awesome main styles minified
  • font-awesome-ie7.min.css: Font Awesome for IE7 styles minified
  • gridex.css: Gridex plugin styles
  • lush.animations.min.css: Lush Content Slider animations keyframes
  • lush.min.css: Lush Content Slider animations and styles

Font Awesome More Styles

Font awesome more is based on 3 css files. Each file has its own set of icon. If you don't need some icons you can remove easily the css file.

Stlyes are included in the main CSS file via the @import sentence.

  • font-awesome-corp.css: Font Awesome More Corporate icon styles unminified
  • font-awesome-ext.css: Font Awesome More Extra icon styles unminified
  • font-awesome-social.css: Font Awesome More Social icon styles unminified
  • font-awesome-more-ie7.min.css: Font Awesome More for IE7 styles minified

Javascript

Files

The theme use the following Javascript files

  • jquery.js: Javascript library required for all plugins
  • bootstrap.js: Bootstrap main Javascript plugin
  • bootstrap.min.js: Bootstrap main Javascript plugin minified
  • html5shiv.js: Add support for HTML5 in older browsers
  • bootstrap-gridex.min.js: Porfolio expandable items for Bootstrap plugin.
  • jqBootstrapValidation.js: Input validation plugin. Used for the contact form.
  • jquery.tweet.js: Plugin for Twitter feeds
  • jquery.gmap.min.js: Plugin for Google Maps
  • jquery.easing.1.3.min.js: Plugin with animations function
  • jquery.lush.min.js: Lush Content Slider. Used as the main page slider
  • application.js: Custom theme Javascript. Customize the theme JavaScript editing this file.
  • contact.js: Containes all funcitons for the form validation plugin. Customize the form editing this file.

Twitter Feed

Use the twitter plugin directly from your markup with the data-username attribute


<p class="tweet-feed" data-username="wrapbootstrap"></p>

Google Maps

Use the twitter plugin directly from your markup with the following attribute:

data-zoom the map zoom. A number from 1 to 20.
data-markers your company address. Add multiple address separated by ;


<div class="gmap"
data-zoom="16"
data-markers="Address 1; Address 2"></div>

HTML Structure

This theme is divided in sections. Each section can be accesed via a menu or simply stay next to others sections (e.g, twitter section).

Each section has its own id that indicate the content they represents.
Sections structure used in this theme: 

<body>
  <section id="sec-slider"> ...
  <section id="sec-navbar"> ...
  <section id="sec-services"> ...
  <section id="sec-clients"> ...
  <section id="sec-portfolio"> ...
  <section id="sec-about"> ...
  <section id="sec-twitter"> ...
  <section id="sec-contact"> ...
  <footer> ...
</body>

Section Classes

By default a <section> has a white background color. To change this you can use the following classes to alternate between color:

  • section-dark: set a dark background color
  • section-alt: set a soft dark background color

Contact Form

The contact form uses a validation plugin jqBootstrapValidation desinged to work with Bootstrap form.

This plugin works automatically on input with a required property and detecting the type attribute to make the correct validation.

To controll user submit action there is a file contact.js which has the plugin invokation and the ajax functions correctly added to send the form data when all inputs validates successfully.

To use your own PHP contact form, you need to change the variable CONTROLLER_URL with the url of your PHP controller.

Theme Font

This theme use the Open Sans font and is included in file index.html in the following way 

<link href='http://fonts.googleapis.com/css?family=Open+Sans:300,400,600,700,800' rel='stylesheet' type='text/css'>

As you can see the font is host by Google Fonts, you can change the url for the font you want using what Google provides
